# Simili Documentation

Welcome to the Simili documentation. Simili is an AI-powered GitHub issue intelligence tool that automatically detects duplicate issues, finds similar issues across repositories, and routes misfiled issues to the correct location.

## Features Overview

### Duplicate Detection
Automatically find and link similar issues when new ones are created, reducing duplicate work.

### Cross-Repository Search
Search for similar issues across all enabled repositories in your organization.

### Automatic Issue Transfer
Route misfiled issues to the correct repository based on labels, title, or content.

### Custom Bot Identity
Use a GitHub App to have comments appear from your organization's bot instead of `github-actions[bot]`.

### Dual Token Support
Use separate tokens for comments (bot identity) and transfers (elevated permissions).
